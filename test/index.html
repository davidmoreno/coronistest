Coronis Test tester

As coronis test is a test framework I dont think is good idea to use itself to test it, nor another test framewok. So The only option left is to create a custom tester.

On this tester there is no jQuery nor any other helper libs, as they may interfere. Just press the main button Test to start the full suite, or a custom test with every button.

For security reason on browsers, its better if you host this somehwere and use it throught a web server.
